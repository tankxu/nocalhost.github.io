Nocalhost 是什么？

一个

连通云端和本地 IDE 的桥梁

开发者可以便捷地在 IDE 中将代码变更快速部署到云端环境。

一个

允许多个开发者共享的开发空间

协作共享，自由无间，云端开发空间可共享或者独享。

一个

Kubernetes 的面向开发的扩展包

Nocalhost 搭配 Kubernetes 可以使得开发者更加便捷的开发基于 Kubernetes 的应用。

一个

照亮微服务开发的聚光灯

开发者可以在复杂的微服务框架下直接将指定微服务转换成开发模式，进行开发和调试。

Nocalhost 是一种新的云端开发方式

Localhost 开发模式

所有服务运行在本地，开发者之间的开发环境不互通。

  • 受限于本地电脑性能，难以开发大型微服务应用
  • 搭建和维护开发环境困难
  • 联调繁琐
  • 开发环境与生产环境差异较大
Nocalhost 开发模式

在 Nocalhost 的加持下，云端微服务架构立即变身为开发者协作联调环境。

  • 云端弹性资源可支撑大规模微服务应用开发和部署
  • 开发环境可以动态按需准备
  • 云端共享环境，方便联调开发
  • 与生产环境差异很小
